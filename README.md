# 또박또박 AI 글씨 코치

초등학생 손글씨 사진을 업로드하거나 촬영하면 AI가 판독하고, 정답 문장과 비교해 가독성 점수와 피드백을 보여주는 웹앱입니다.

## 포함 기능

- 사진 업로드와 카메라 촬영
- 원본 이미지 미리보기
- Gemini 2.5 Flash 기반 손글씨 판독용 Netlify Function
- 정답 문장과 글자 단위 비교
- 불확실한 글자 강조
- 선생님 피드백 메모
- 학생별 제출 이력 저장
- 반 전체 통계
- 인쇄 대화상자를 통한 PDF 저장

## Netlify 설정

Netlify 환경변수에 다음 값을 등록하세요.

```text
GEMINI_API_KEY=your_gemini_api_key
```

배포 설정은 `netlify.toml`에 포함되어 있습니다.
